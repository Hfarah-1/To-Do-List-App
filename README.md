# To-Do List App

![Swift](https://img.shields.io/badge/Swift-5.9-orange?logo=swift&logoColor=white)
![Platform](https://img.shields.io/badge/platform-iOS-lightgrey)
![License](https://img.shields.io/badge/license-MIT-blue)

A simple and clean **To-Do List app** built with **SwiftUI**.  
This app allows you to add tasks, mark them as completed (with strikethrough text), and delete them with a swipe gesture.  

---

## Features

- ➕ Add new tasks  
- Mark tasks as completed by tapping them  
- Delete tasks by swiping left  
- Built with **SwiftUI** for a modern and responsive interface
